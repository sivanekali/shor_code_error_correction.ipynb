# 9-Qubit Shor Code – Quantum Error Correction using Cirq

This project demonstrates encoding of one logical qubit into nine physical qubits using
the 9-qubit Shor Code. Single-qubit errors are detected using syndrome measurements.

## Tools
- Python
- Cirq
- NumPy
- Scikit-learn

## Dataset
Iris dataset (used minimally)
